# Lenovo-P340-Tiny-Hackintosh
基于opencore0.7.8制作，支持Monterey12.2.1及其它低版本的系统

机型：Lenovo p340 tiny

主板芯片组：Intel q470

cpu：Intel 10500 65w

内存：三星ddr4 2666 8g*2

独立显卡：amd rx560 4g

集成显卡：Intel uhd630

无线网卡：bcm94360cs2

固态硬盘：wd sn550 500g

显示器：LG 27UL850(DP输出)

本efi纯属自用，USB已定制，配置和我类似的朋友可自行取用，不保证一定能够正常运行，下载后请自行添加三码

功能实现

休眠&唤醒 正常（DP睡眠唤醒均正常，HDMI睡眠唤醒后黑屏）

隔空投送 正常

独立显卡 正常

igpu核显加速 正常
